# Проектная работа: "Mesto". Создана в рамках учебы в [Яндекс.Практикум](https://praktikum.yandex.ru/) на факультете ["Веб-разработчик"]

## Описание: 

"Mesto" - это типовой однастраничный сайт. В проектной работе была применена работа с макетом в Figma, раcширенные возможности HTML и CSS и реализация адаптивной вёрстки посредством Flexbox, Grid Layout и Media queries. 

## Функционал: 

* Адаптивный дизайн 
* Трансформация элементов страницы посредством CSS
* Редактирование профиля посредством JS

## Технологии: 

* Flexbox 
* Grid Layout 
* Positioning Elements 
* BEM Methodology 
* File Structure and File Paths (Nested BEM) 
* Git 
* Figma 
* JavaScript
